1. 激励规则与分发方案
(1) 激励规则的初步讨论
- 基础奖励：年化 5% 的基础奖励，持续时间三个月
- 奖励维度： 
  - 用户持仓时间：可设定最低持仓时间（如一个月或一个季度）后才分发奖励。
  - 用户行为奖励：增加 LP、购买特定资产、抵押等行为可获得额外激励。
- 灵活性： 
  - 激励规则可以根据需求调整。
  - 不强制要求用户将 Token 投入 Dex 或其他协议。
(2) 与链上 DeFi 的合作
- 合作对象：HyperIndex、DoDo 等链上 DeFi 项目。
- 合作模式： 
  - Dex 可提供的交易手续费分红将作为额外激励。
  - Dex 或 Lending 协议能否提供额外支持（如流动性挖矿奖励）需进一步沟通。
- 用户收益结构：基础奖励 + DeFi 分发奖励。

- 活动页面：需开发前端展示页面，用于用户查看规则和自身收益进度。
- 开发周期：前端页面与脚本开发预计需要 1周 时间。
- 排期：需将开发任务纳入内部产品开发排期中。
激励发放模式
基础年化激励（1%） + 行为激励奖励（4%） + DeFi项目额外激励（optional）

(1) 基础年化奖励
- 规则：用户持有活动指定的资产（如 XAUM ），持有一定时间以上可以拿到 年化 1% 的基础奖励。
- 最低持仓天数： 
  - 持仓时间需达到30天以上才能获得对应的奖励。
- 奖励计算公式： 
基础奖励 = 用户持仓金额 * (1%年化收益) * 实际持仓天数 / 365

(2) 行为激励奖励
- 规则：用户在链上的 DeFi 协议中完成以下行为，可获得额外奖励：
  - 增加 Dex 流动性（LP）： 
    - 用户将资产注入 Dex 的 XAUM-USDT/USDC 或 XAUM-HSK 交易池，提供流动性。
    - 奖励按 LP 持仓时间和金额分发。
  - 参与 Lending 协议： 
    - 用户将 XAUM 存入链上的 Lending 平台作为抵押资产。
    - 奖励按抵押金额和时间分发。
  - 交易行为奖励(由项目方提供，和DeFi项目确认)： 
    - 用户在指定 Dex 上完成一定数量的交易，可获得小额奖励（例如DEX手续费返还等）。
- 奖励计算公式：
行为奖励 DEX = 提供DEX XAUM LP * (4%年化收益) * 实际天数 / 365
行为奖励 Lending = 存入抵押资产 * (4%年化收益) * 实际天数 / 365
